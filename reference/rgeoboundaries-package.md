# rgeoboundaries: A Client for the 'geoBoundaries' Administrative Boundaries Dataset

Access data from the 'geoBoundaries' API
<https://www.geoboundaries.org/api.html>, which provides country-level
political and administrative boundary datasets. See Runfola, D. et al.
(2020)
[doi:10.1371/journal.pone.0231866](https://doi.org/10.1371/journal.pone.0231866)
.
